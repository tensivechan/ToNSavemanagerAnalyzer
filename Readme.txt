ToNSaveManager Analyzer

ToNSaveManager の保存データ JSON を読み込み、解析結果を見やすく表示する Windows 用アプリです。

使い方
1. `ToNSaveManager Analyzer.exe` を起動します。
2. 解析したい JSON ファイルを選びます。
3. 必要に応じて RoundType、並び順、検索で絞り込みます。

表示できる主な項目
- RoundType
- TerrorID
- Terror Name
- MapID
- Map Name
- Player Count
- TD の番号

配布物について
- `ToNSaveManager Analyzer.exe`
- `Readme.txt`
- 設定ファイル類をまとめたフォルダ

注意
- データは複数ファイルまとめて読み込めます。
- `(blank)` のデータは非表示にできます。
- `Content` や `Result` は表示しません。
